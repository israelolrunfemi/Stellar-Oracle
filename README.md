# Stellar Oracle

A price oracle for the Stellar network. Fetches real-world asset prices and
writes them on-chain via a Soroban smart contract.

## How it works

Internet prices → TypeScript aggregator → Stellar chain

## Quick start
```bash
npm install
cp .env.example .env
npm run dev
```
